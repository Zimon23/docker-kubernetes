
- 쿠버네티스 핵심 컴포넌트 실행 목록 확인 
    ```
    kubectl get -pods --namespace kube-system
    ``` 
- 쿠버네티스에 등록된 노드 확인 
    ```
    kubectl get nodes
    ```
- kubectl의 도움말 확인 
    ```
    kubectl -h
    ```
- kubectl 명령어(get)의 도움말 확인 
    ```
    kubectl get -h
    ```

- kubectl 노드가 연결을 끊을 때 또는 삭제 할 때 사용하는 명령어 
    쿠버네티스 master 설치시 다시 설치할 때도 사용합니다
    ```
    kubectl reset 
    ```
