
- 쿠버네티스 핵심 컴포넌트 실행 목록 확인 
    ```
    kubectl get -pods --namespace kube-system
    ``` 
- 쿠버네티스에 등록된 노트 확인 
    ```
    kubectl get nodes
    ```
- kubectl의 도움말 확인 
    ```
    kubectl -h
    ```
- kubectl get의 도움말 확인 
    ```
    kubectl get -h
    ```

- kubectl 각 노드 삭제   
    ```
    kubectl reset 
    ```
